How do I start mining on DabMining?

Follow these steps to get started:
  • Download whatever software you'd like for CPU, GPU, or ASIC mining.
  • For the workername, simply use the address of whichever coin you are mining.
  • Do not use addresses from exchanges to mine to! Control your own wallet addresses.
  • For the password, you can use any password or leave it blank.
Here is an example setup for your miner:
  • cgminer -o stratum+tcp://stratum.dabmining.co:9994 -u RR9CAfG7pxvcE41hicHmUMpmpPqfApe5p3 -p dabmining
